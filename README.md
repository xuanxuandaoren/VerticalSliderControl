# VerticalSliderControl
一个滑动的自定义控件，思想不错，可以参考下
##使用效果
#####1、可以通过滑动来控制控件显示的等级
##注意事项
#####1、使用时要把项目中res/layout下面的verticalslider.xml拷贝到自己项目的res/layout项目下
#####2、使用时把项目中的res/drawable和res/drawable-xhdpi项目下的图片拷贝到自己项目下对应的目录中
#####3、把项目中的VerticalSliderControl拷贝到自己项目中就能使用了
##使用方法
#####1、获取控件对象，调用对象的setOnSliderTouchListener就可以为对象设置滑动监听
#####2、获取控件对象，调用对象的setProgress就可以代码设置控件的高度
